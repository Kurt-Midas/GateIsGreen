Hello Alpha Tester / Prospective Partner!

Gate Is Green is a fleet management tool. It allows you to view fleet member information in the browser in various helpful views.

The tool is under active development and your feedback / feature requests / bug reports / code reviews are critical to its success. Please contact me through Github, Twitter, or at my character "Kurt Midas"

I'm more than happy to help you deploy a non-public version.

How to Use

  1. As fleet boss, select "Copy External Fleet Link" and paste the NUMBER into the box above
  2. The SSO button will redirect you to the official CCP SSO site. Log in and give the app
    • ONLY the character in charge of the fleet
    • ONLY fleetRead and fleetWrite permissions
  3. You will be redirected to the fleet management page. Only share the link with those you trust.

Concerns

  • Your link may become invalid at any time without warning. A crash bug will cause the app to restart and state is not preserved between launches.
  • Do not use this for important fleets. I cannot guarantee no security bugs.
  • Anyone with the fleet link can access the fleet. The three ways to invalidate a link are to disband the fleet, change the fleet boss, or use the Eve service site to remove this app's permissions
  • Do not use important characters. If I changed the code to extract your refresh token I could theoretically access any future fleet on that character. Use the Eve service site to remove this app's permissions
  • Most information is on a 30 second cache. I cannot control this.
