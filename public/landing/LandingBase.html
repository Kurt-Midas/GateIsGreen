Hello Alpha Tester / Prospective Partner!

This tool is called "Gate is Green" and uses the new Fleet API. To Summarize, it will...

  • Come in an easy-to-deploy form so you can run your own
  • Allow an FC to create "Read" and "Write" links to share
  • Allow both types of link to:
    • See various aggregations of fleet member info
    • See a display with sortable and filterable information
    • See a display with fleet structure, including locations and ships of various wings
    • See alerts whenever pilots join/leave/move around a fleet
    • See alerts whenever a pilot changes ship type
    • See updates of this information in real-time as the CREST cache allows
  • Additionally allow anyone with a "Write" link to:
    • Invite pilots to [specific positions in] fleets
    • Boot people from fleets
    • Move fleet members to wings/squads/roles

You may notice it does very few of these things right now. Work in progress

You can help development by providing help offers / bug reports / feature requests / feedback / exotic dancers to my character Kurt Midas

How To Use

  1. Form a fleet
  2. Click the upper left menu and scroll down to "Copy External Fleet Link"
  3. Paste the number (JUST THE NUMBER, I'll fix that later) into the box below and click the SSO button
  4. This will redirect you to the official CCP SSO site. Log in and give the app
    • ONLY the character in charge of the fleet
    • ONLY fleetRead and fleetWrite permissions
  5. You will be redirected to the fleet management page. The link is unique and sharing that page will give full access to your fleet's information. Currently there is no required additional authentication, notifications that another user has connected, or means to boot users (in an emergency please use the Eve service site to revoke my app's SSO permissions. The next refresh will fail, deleting the session and invalidating the link, and a new link can be created).

On an important note, this app might have security bugs and you don't know and shouldn't trust me. Using this tool with stratops or move ops is a very bad idea.
